# Instrucciones para el archivo APK

Coloca tu archivo APK de CarGo aquí con el nombre `cargo.apk`

Si quieres usar un nombre diferente, actualiza la constante `DOWNLOAD_APK_URL` en `src/consts.ts`

## Ejemplo:

```
public/
  downloads/
    cargo.apk  ← Tu archivo APK aquí
```

Una vez colocado el archivo, el botón de descarga funcionará automáticamente.
